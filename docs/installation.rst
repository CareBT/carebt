Installation
============

From PyPi (recommended)
-----------------------

.. code-block:: bash

    pip install carebt

.. important:: 

    Currently pip install does not work, as carebt is not published on PyPi. This will be done soon.

From source
-----------

.. code-block:: bash

    $ git clone https://github.com/careBT/carebt_core.git
    $ cd carebt_core
    $ ./build_and_install.sh
